为什么总是来不及反应

发表于 2020-08-05 分类于 定投实践 阅读次数： Valine：


人类的思考是建立在逻辑之上的，如果不是将和动物没有区别，我们的大脑是为生存而设计的，不是为了思考设计的，如果没有深入的逻辑的思考，将于现代的社会脱离，沦为焦虑、浮躁、碌碌无为的大多数人

未经训练的大脑

无论是在生活中，还是在工作和投资中，无论是人生中终大的事情，还是一些小事，如果不认真的思考，分析，很容易做出不正确或者不是更好的判断

如何才能做到正确的判断呢，需要有思考力，也就是思考能力，在四年前，我写过一篇《恶斗大脑记》，记录了当时思考时大脑中各种各样的嘈杂声音，那个时候，就是几乎完全没有思考力的，大脑以一种自然的不受控制的方式工作，完全和主意识相反，肤浅、焦急、懒惰，而当时我能想到的方法就是和大脑极力抗争，但并解决不了问题

我们的大脑不重要的，经过训练的大脑才重要

思考力的基础不是天然就在原装的大脑中的，因为大脑是为了求存的，不是为了求真的，意思是 亿万年前我的的祖先为了在纷繁复杂的丛林中找到可以食用的果子，找到可以延续后代的生殖资源，这些就像电脑的BIOS一样深深地刻在我们的基因里，以至于我们对生存相关的信息异常敏感，这种求存的感知让我们对表面好的东西充满了敏锐和渴望，就像在人群中很容易发现美丽漂亮的人，就像在菜里很容易尝出脂肪的味道，

而现代的人类世界，是建立在科学、逻辑、制度等非自然生存环境之上的，现在人类解决了绝大多数的基本生存问题，比如饥饿、医疗、居住等等，与自然界的动物相比，人类拥有的一切都是奢侈品，而另一方面，人类构造了一个复杂的，存在于想象当中的社会，那里充满危险、残暴、不公，我们不大可能走在路上被更强大的人捕猎或者吃掉，但是我们的时间、精力、力量却无时不刻的被更强大的思考力捕获或者吃掉

比如我们的注意力，被社交媒体、即时娱乐等软件吸引，我们的钱财，不断地作为韭菜被割，我们的思维空间，不断地被垃圾信息所填充，我们会进入强大者的陷阱，我们会上当受骗，我们会将自己的生命不当回事

这就是我们原始的大脑不能适应现代社会的现象，这些原本能帮助我们不断繁衍，将基因保存至今的方法，却不在适应现在的社会了，因为现代的社会积累只有短短的数百年，简直可以忽略不计，因为任何可以写入基因的认知突变至少需要上万年的时间(物演通论–王东岳)

一个有趣的大脑模型

问题来了，我们怎么才能在现代的社会中生存呢，或者生存的更好呢？有办法，几乎人人都可以，那就是改变自己的认知，做大脑的主人，而不是被原始的大脑带着跑

柏拉图在两千多年前，对人的大脑做了一个至今都想当正确的假设：我们人就像是一辆马车，被一批黑马、一批白马和一个车夫所驾驭。

  • 黑马就是我们的基因，时间最久，力量最大，它控制着我们的心跳，血压，等非意识功能，我们几乎无法感知更不能控制

  • 白马就是我们的情绪，我们能感知，但常常无法控制，情绪一来什么理性科学逻辑，都靠边站

  • 车夫才是我们的意识，我们的理性思维

对于一个没有训练的大脑来说，这辆马车就是有黑马控制，反应迅速，力量强大，横冲直撞，当遇到危险立即逃跑，当遇到利益立即奔赴，很多时候我们由于反应不过来，做了一些事情，我们自己都不知道发生了什么，比如我们突然收到惊吓，慌张的表情、扔掉的东西……

经过训练的大脑，这三种角色会相互配合，遇到一件事情，三个角色各自处理一遍，黑马想立即处理，这是白马做下安慰，白马虽然不会立即行动，但会产生情绪上的波动，这时需要车夫处理以下，看看情况到底是什么样的，于是车夫给白马说，不要难过，看起来熊市是坏事，但是确实我们积聚自己能量的好几乎，应该高兴才对，于是白马转念一想，确实是呀，再去给黑马说，不用着急，不用慌张，这对我们有利的情况

不断地经过这样地训练过程，最终三个角色相互配合，使马车不断向正确的方向前进，所以遇到事情，先停几分钟，让车夫得到消息，是最好的处理方式

车夫如何判断呢，就需要用到逻辑工具，做推理，识别谬误，看到未来，认识影响。

怎么学习和锻炼呢？从一点一滴做起，踏踏实实的学习，实践和总结，没有其他方法，我想你能体会到了，这个过程就是个定投过程，投入的是时间和精力，得到的是自己的能力

稻草人谬误

今天了解一个谬误 —— 稻草人谬误

定义是：通过夸张、扭曲、甚至捏造出一个观点，来使别的观点显得更正确

举例：
A：我应该多做一些大脑训练
B：难道你不用生活了吗？不要管其他事情了吗？
A：我没有说不管其他事情呀
B：难道其他事情就不用时间了吗？就不需要思考和精力了吗？你这样会偏废的，成功不了
A：……

B 就是在用稻草人谬误，他从 A 的话中，引申出了其他观点，用话中的所谓反面，来确立自己的观点，然后得出自认为对的结论

仔细想日常中的这样情况太多了，特别是在家人之间、夫妻之间，因为没搞清楚一个问题的内涵和外延，胡乱扩展，以偏概全，最终没有好结果

如果 A 识别不了 稻草人谬误，可能真的觉得 B 说的有道理，可能放弃了原本计划好的事情，将来的损失不可估量

对于 B，如果意识不到自己的的谬误认知，可能会绝对自己有理，从而也使自己的观点从正面被扭曲，形成认知偏差，他也不会去做 A 所说的事情

总结

  • 认识是思考的基础，有效的思考会带给我们幸运和美化，无效的思考将使我们厄运不断，而且还不知道错在哪里，
  • 我们的大脑由三部分组成，分别黑马、白马、车夫构成，需要不断地锻炼使三者相互配合
  • 稻草人谬误，会使我们以偏概全，曲解别人的观点，扭曲自己的观点
