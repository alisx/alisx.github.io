神器 Pypandoc —— 实现电子书自由

发表于 2021-09-16 分类于 python 阅读次数： Valine：

知识就是力量，甚至说 知识就是金钱 更为合适。

在信息时代的我们，获取知识，提升认知如同吃饭一样，成为了必须做的事情，阅读是最常用的获取知识的方式，没用之一，可以有时因为一些技术问题，导致我们阅读不便。

比如前几天，我想读一本 《人人都能用英语》，找到了在这里找到了用 Markdown 写的原文，虽然可以阅读，但如何在手机上阅读，是个问题。

转为 Pdf，或者 html，可以是可以，不过阅读起来不方便，也记不住上次读到哪里了，更要命的是无法做笔记，也搜了纸质书，因为这本书没用正式出版，所以也都是盗版的。

因为我经常用 多看APP 读书，如果能直接在多看上看就好了。

尝试了以一圈，发现了一个好方法，现在我以及阅读过半了，现在分享一下我的魔法过程。

解题思路

首先看看多看支持什么样的电子书格式，百度一下，特别全，为了更好的体验，还是转为 epub 更好。

因为 PDF 没法自适应行，只能通过放大缩小控制。

那么如何转 epub 呢？一搜，发现有众多在线转的，尝试了几个，不是收费就是各种限制。

咱不是会编程吗？如果能自己搞岂不更好。

改变检索思路，很快发现了一个命令行工具 —— pandoc，进一步研究，发现竟然有 Python 版的 —— pypandoc，真是太棒了，研究一番，发现完全满足我的需求。

于是解决方案就是：

  1. 下载 markdown 版电子书
  2. 用 pypandoc 转化为 epub 格式
  3. 导入 多看 阅读

像回家吃饭一样

发表于 2021-09-06 阅读次数： Valine：

“只要是占便宜、免费的，一点都不碰”；

“看到送餐小哥找不见商家，就会主动上前帮忙”；

“旅游呀，就是受罪，还是每天为大家做点事情实在”；

“还用锻炼吗？每天来回走能有十多公里了”

……

如果不是亲耳听到，我一定不会相信，这些话出一位餐厅大妈老板之口。

1

自从公司搬到五路居后，经常去附近地下商场的美食城吃饭。其中一家干净整洁，风格优雅的餐厅 —— 6号拌面，很引人注意。

商场通道里的他们海报很有特色，除了菜品更多的是与食客的合影。

每次经过门口，都会被老板的热情张罗所感染。偶尔和同事去过几次，都是点菜，感觉还不错。

2

后来，因为我饭点不定，常一个人去吃饭，于是再去试探了一下，这才知道除了点菜，还有现成的红碗、绿碗两大类菜品可直接选。

红碗荤菜多，贵些，绿碗素菜多，便宜点，每种都有多样可选，另外主食有米饭，炒饭，面条，都是免费不限量的。我比较喜欢吃红碗的红烧鸡块，和绿碗的卤肉豆角。

自此之后，每天中午都去那里吃。

有天拿了碗红烧鸡块，吃完结帐时才意识到是绿碗，问店员是否装错了，店员说：今天立秋，天气转凉，老板嘱咐所有菜都用绿碗盛，作为 立秋特供。

3

几天后，我忙完已经下午2点多了，去吃饭，店里没客人，大妈正在和店员们吃饭。

我选了一份菜，边吃边聊了起来。虽然在这吃了几个月了，但没怎么聊过。

我提到那个立秋特供时，打开了话匣子，聊了很多，于是听到了前面的那些话。

她说，有次在离这里很远的一个商场，拎了一堆东西，突然一个小伙子跑过来说：“ 阿姨，您怎么跑这么远买东西呀，来来来我帮您拿…… ”

这个小伙子是她之前帮助过的一个送餐员，当时她已经不记得帮过他了。

她说，这个世界上，只要你做好事，帮助别人，迟早会还回来的。

像这样的事情在她身上发生的太多了。

她递给我一份菜单，我才发现这个菜单不一般 ——

每页，一面是菜品，另一面是自己写散文诗，最后几页还有易读错字表

散文诗，有的是四个季节的描述，有的是旅途中的感悟，用词优美，并糅合古诗词，一种愉悦感跃然纸上。

其中有篇在去年疫情期间写的文章，让我印象深刻，用拟人的手法，讨问病毒，更展现了人民战胜困难的勇敢与坚毅，以及表达了 对未来的信心和祝愿。

4

为什么要做这样的菜单呢？

她说，可以让客人在等菜时看看，调节一下心情，舒缓一下情绪，让客人感觉像回家吃饭一样。

虽然这里菜上的很快，客人也都只看手机，但这一份用心，蕴含无限温暖。

每次，我都能感受到，回家吃饭的踏实、从容与温暖。

5

不占便宜，踏踏实实，认认真真，干干净净

是一种修行，是一种的习惯，更是一种生活的态度
