灵感可以制造

发表于 2016-05-23 更新于 2020-07-24 分类于 开启心智 阅读次数： Valine：
灵感是个奇妙的东西，她能帮助我们解决思考了很久的问题，能给我们带来好运，如果你是从创作者，灵感简直就是天神，她赋予我们超乎正常了能力的水平，让我们做出一个个神作。不过人们普遍认为灵感可遇不可求，所有才有江郎才尽的感叹。其实**灵感并非乍现，是有规律可循的**，而且我们能学会如何更好的利用她。
**灵感是怎么形成的** ？灵感的产生和大脑的工作方式有关，大脑可以分为有意识和无意识两部分， 有意识部分，比如思考，演讲，学习等，是我们有意识的。无意识部分又可以分为身体机能协调部分和未知部分，身体机能协调部分主要用来控制内脏器官和内分泌等。而未知部分占据了大脑的很大一部分，就像我们对海洋的了解一样，这部分在不断的活动，比如做梦，走神，胡思乱想等，灵感也是这么来的，这部分的大脑不断的将一些独立的神经元连接起来，形成新的信息，就像个调皮的孩子，将东西不停的胡乱的摆放，突然发现有种摆放法很好，就像灵感产生一样。所有灵感也被我们称作闪念。
为什么说神经元关联起来有用呢？最近科学界有项研究结果，说人类大脑和大猩猩的并没有太大差别，但人类的大脑中有种叫髓磷脂的物质比大猩猩多很多，它可以像橡皮管一样将神经元间的链接包裹起来，使神经元之间的信息传递速度提高3000倍，而且爱因斯坦的大脑中的髓磷脂的含量高于普通人。所以让神经元之间建立更多的连接，我们就会更聪明，想法更灵活，也就是智商更高。
如何让灵感产生呢？只要不断的将不同的神经元连接就行，但我们除了学习和反复练习之外几乎没有办法实现神经元之间的连接，只能寄希望于灵感了。我们知道灵感是由于大脑的未知部分制造的，那么如果大脑是台电脑，想让他它更快的完成某个任务，最简单的做法是关掉其他无关的任务。所以要做的是**让大脑休息**。你可能有这样的体验，有个问题百思不得其解，决定放弃了，出去放松了一下，突然有了解决思路，这就是让大脑从高度费脑力的事情上解脱出来，形成的灵感。
除了让大脑休息，还需要做一件事，就是让大脑中的神经元有足够的信息，如果被链接的神经元中并没有信息的话什么都不会产生。相对于让神经元连接让其装入信息更容易，做法就是**多看多读多听多想**，不要计较收集到的东西是否对当前有用，因为当他们联系起来是是知道谁产生什么神奇的东西，这样将不同的东西联系起来产出新的东西的例子太多了，所有我们只管收集，也不要担心忘掉，忘掉只是你主意识找不到了，但我们的神经元会将信息保存很久(电影《头脑特工队》里有很形象的描述)。
所有让灵感产生很容易，收集各种信息，然后去休息就好了。
有了灵感产生的条件，我们该如何更有效的利用她呢？有人会说，都是灵感了还能放过吗，一定会好好利用的。其实不然，拿破仑希尔有次和朋友 出行， 途中看到他的朋友不时的在小本上写些东西，等到了目的地他问朋友，您还记得刚才在本子上写 了什么？他的朋友一脸茫然的说不记得。灵感之所以被叫做闪念，就是说明她会一闪而过，几乎抓不住。
利用灵感的做法是：
  1. 记录灵感，可以随身带个小本子，有了想法立即记下来，或者做个录音记录，一般手机上都有录用功能（别找借口噢）。
  2. 寻求外部刺激，比如和朋友聊天，看看电影什么的，很多时候想的太深可能陷入死角，百思不得其解，这时别人的一句话或者一个动作可能会提醒你，让你就想到解决的办法；还有别人可以能站在和你完全不同的角度上思考，能给你带来其他维度的视角
  3. 记录聊天内容，现在的通信工具很多，聊天时最好使用文字，这样不但可以锻炼自己的描述能力，重要的是可以将聊天的内容记录下来，一旦哪天想起曾经说过的东西，可以方便的翻看；
  4. 将记录的东西定期翻看，将平日里点点滴滴的想法汇聚起来，不断的强化，不断的再思考，建立新的链接，形成更好的思想或方法，帮助我们做出更好的创作。唐代诗人李贺就是这么干的，他每次出行时就将看到的想到的记录在小纸条上，（当时没有圆珠笔可想多么痛苦），回到家中将这些纸条翻看、梳理，写成一首首流传千古的诗句。
