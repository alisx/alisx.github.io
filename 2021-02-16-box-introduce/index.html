BOX —— 数字货币的高速列车

发表于 2021-02-16 更新于 2021-03-22 阅读次数： Valine：

今天聊聊什么是 BOX

这个 BOX 不是盒子，而

是由李笑来设计发行的基于数字资产的ETF基金

李笑来

️别一看李笑来就怕被割韭菜，稍微看一下这么多年了数字货币跌宕起伏，无数曾经所谓的"正义代言人"跑路的跑路，消失的消失，只有李笑来不仅没有消失而且比以往更活跃

不断地写书，开课，成立投资社群…

帮助更多的人走出迷茫的状态，获得财富自由

当年(17年)各种空气币肆虐，坑钱无数，国家开始严打整治，那些妖言惑众、借币行骗的小丑们，纷纷出逃国外避难，朋友们劝刚好正在日本的李笑来，继续待在日本避避风头，不要回来，可他立即回国，直面问题，一直到现在，依然呆在北京，继续做着改变世界的事情

就在前几天，他宣布，用自己投资的375万个 BOX，成立基金会，在我们的大西北种树，为我们的大地母亲做点事情(正在招募生态治理、植树造林方面的专家，如果您正好是，请及时与我联系)

看到这些，请问到底是谁在割大家的韭菜

如果还有疑虑，请找资料研究下，毕竟做无论做什么不能只听一面之词哈

接着说BOX，BOX 是由以 比特币 为主的几种数字货币组成的数字货币基金，通过以太坊的智能合约发布，以资产凭证（token）的形式交易

比特币

️别一看 比特币 就说涨的太高了，现在入场来不及了

如果是投机的话还真有点晚，因为涨的太高可能意味着一个很长时间的低迷，被套住可不好受

但是采用定投策略的话，看的是趋势，不用我展示，随便找个比特币价格波动图，就能看到趋势

另外，这里特别提醒下:

如果有人对你说: 比特币你已经错过了，千万别错过XXX，十有八九是骗子

可以这么想: 苹果，可口可乐，就这么一个，长期投资价值巨大。

如果实在不行，等等看他推荐的在未来一段时间使用的人会不会变多，那时再做决定也不晚。

BOX 的特别之处

BOX 与一般基金不同的是

具有零管理费，零开润 的特点

意味着什么呢？

简单说，BOX 基金的设计者获利的唯一方式就是和其他投资者一样:

只能靠投资 BOX 获利，无法通过抽成和管理费获利，相当于设计者将自己的利益和大家绑定在一起

如果还不理解，随便看看市场上的基金，有几个基金经理是会投资自己管理的基金的？

更别说 零管理费，零开润了，因为世界上，目前只有 BOX 具有这个特点

另外，BOX 的成分会根据实际市场情况做出调整，从开始的比特币，EOS和Mixin，三种成功，调整为目前的七种，就像航船，需要不断地调整方向，方可顺利地达到彼岸

好了今天就到这，比心
