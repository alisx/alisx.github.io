牛年，牛礼物

发表于 2021-02-14 更新于 2021-03-22 阅读次数： Valine：

说着容易，做着难

我们都知道，最好的赚钱方式是用钱赚钱，具体方法是投资

可事实上并不容易做

  • 选择什么标的(股票，基金，比特币…)
  • 采用什么方式(定投，一把梭，高卖低买…)
  • 无常的意外(轮胎爆了，牙坏了，发红包了…)
  • …

无数事情都会影响我们，导致投资失败

本来我们的注意力资源就不够，选择力更不足，去做投资就好比直接扔进社会孩子 —— 不知所措

虽然想要投资，获得更多的收益

但是，生活还得继续，家还得养，工还得打，钱辛苦点还得挣

至于什么投资，什么增长，去它的，但求没有损失，若能有点收益就烧高香了

于是我们除了 埋头苦干，出卖时间和精力，别无它法

其实这样也对，毕竟投资市场是鱼龙混杂，陷阱无数，于其费力不讨好，还不如安分守成

缺什么

难道自己真的和钱赚钱无缘，只能望洋兴叹吗？

实际上并不是，只是:

缺少一些必要的知识，缺乏一个锻炼的机会，缺失一个共同成长的环境

我的经历

在去年（2020年）8月份，我接触到了 BOX定投践行群，参与了BOX定投活动

半年左右的时间里，经历了几个月的平淡，元旦左右的飞涨，紧接其后的腰斩，又迎来了飞速上涨

这个过程简直就像过山车，收益率最低时不到 2%， 最更时超过 160%

我之所以没被甩下去，完全是因为我在 BOX 定投践行群里，可以得到不断的心理按摩，学习逻辑知识，投资知识，锻炼写作能力，就像乘坐头等舱，提供实时安全提示的同时，还有更多的附加服务

这段时间里，场外（BOX 定投践行群外）一片醉生梦死，鬼哭狼嚎，而场内，一片安宁，大家该干啥干啥，市场的巨大波动对我们毫无影响

BOX 定投践行群

BOX 定投践行群的投资标的就是 BOX

BOX 是由以 比特币 为主的几种数字货币组成的，数字货币基金

与一般基金不同的是，具有零管理费，零开润 的特点

意味着什么呢？

简单说，BOX 基金的设计者获利的唯一方式就是和其他投资者一样

只能靠投资 BOX 获利，无法通过抽成和管理费获利

相当于设计者将自己的利益和大家绑定在一起，如果还不理解，随便看看市场上的基金，有几个基金经理是会投资自己管理的基金的？

更别说 零管理费，零开润了，因为世界上，只有 BOX 具有这个特点

BOX 践行群除了提供各种知识和课程之外，最重要的是以下几点概念

  1. 投资的理念
    投资并不是投资波动，比如高卖低买（并不是说这样的方式不对，只是对于大多数普通人来说不适合），投资的是趋势，趋势不是短的时间里就会体现的，它需要一个长的时间区间才能看到

  2. 定投策略
    投资方式多种多样，定投是最简单而且最有效到的，简单说，定投就是定期，定额，投资一个标的的投资方式之所以有效，是因为无论什么市场，熊市的长度，总是远大于牛市定投相当于在漫长的熊市里，积攒筹码，在熊市里展现价值

  3. 投资标的
    投资标的是数字货币，对于一般人来说，一听说虚拟的东西，就觉得不靠谱，可以我们人类社会就是建立在一个虚构之上的，所谓的现实与虚拟，不过是感觉上而已只要稍微了解一下数字货币，比如比特币，就能知道它的安全性比其他金融产品都高，因为它不可篡改，不能增发，无法抵赖随着时间的推移，越来越多的我们熟悉的资源于其挂钩，这段时间（2021年初）不断的有超级企业加入数字货币的队列，比如特斯拉、苹果等因此长久看，增长趋势明显

一切复杂的变化多端的都只是表象，投资的本质是看好未来的趋势，操作方法是定投，不断跟上趋势的步伐。

也许你觉得不过是个理财嘛

那么当你知道社群的理念是:

读书锻炼帮朋友，成长投资陪家人，

就能体会到

所谓的投资是对自己的全面提升

我也不是一开始就接受和认定
